# Credit Scoring Model

## Overview

This project is a Machine Learning-based Credit Scoring Model developed as part of the CodeAlpha Machine Learning Internship.

The model predicts whether a customer is creditworthy based on financial factors such as income, debt, and payment history.

## Features

* Data Loading and Preprocessing
* Data Visualization
* Logistic Regression Model
* Creditworthiness Prediction
* Model Evaluation

## Technologies Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Scikit-Learn

## Evaluation Metrics

* Accuracy
* Precision
* Recall
* F1 Score
* Confusion Matrix

## Dataset Attributes

* Income
* Debt
* Payment History
* Creditworthy (Target Variable)

## Project Workflow

1. Load Dataset
2. Perform Data Analysis
3. Visualize Data
4. Train Logistic Regression Model
5. Evaluate Model Performance
6. Predict Customer Creditworthiness

## Result

The model successfully predicts whether a customer is creditworthy based on their financial information.
